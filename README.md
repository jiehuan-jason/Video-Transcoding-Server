# Video Transcoding Server

将上传的视频转码为240p AAC_LC伴音

## 测试环境

Nginx v1.23.3

Node.js v20.17.0

## 使用前配置

 请在server.js里配置好端口，并在index.html中输入server的ip（域名）和端口，如果在同一机器上请**不要**使用localhost或127.0.0.1

使用nodejs启动server，并把index.html放到你的服务器目录内（如apache和nginx），然后Enjoy it！
